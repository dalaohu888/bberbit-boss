# rocboss-frontend

> ROCBOSS Web Frontend Project.

## Build Setup

``` bash
# install dependencies
$ npm install

# build for production and launch server
$ npm run build

# serve with hot reload at localhost:3000
$ npm run dev

# launch server
$ npm start
```
